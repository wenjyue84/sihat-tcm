# Sihat TCM

**AI-Powered Traditional Chinese Medicine Diagnosis System**

Sihat TCM is a cutting-edge digital health platform that combines ancient Traditional Chinese Medicine (TCM) wisdom with modern Artificial Intelligence. The system provides accessible, personalized, and multi-lingual health consultations anytime, anywhere.

## Key Features

- **Four Pillars Diagnosis**: AI-powered analysis using Observation (Tongue/Face), Listening (Voice), Inquiry (Chat), and Palpation (Pulse).
- **Personalized Reports**: Detailed TCM recurrence, pathomechanism, and holistic recommendations.
- **AI Meal Planner**: Personalized 7-day meal plans based on TCM constitution with shopping lists and food suitability checker.
- **Snore Analysis**: Sleep sound recording and analysis for snoring and sleep apnea indicators.
- **Vitality Rhythm**: Meridian Organ Clock, 24 Solar Terms guidance, and seasonal health tips.
- **Qi Dose & Qi Garden**: Gamified TCM exercise routines (Baduanjin, etc.) and virtual herb gardening.
- **Circle of Health**: Constitution-based anonymous support communities.
- **Family Care**: Manage health profiles for the entire family.
- **One-Click Remedy**: Integrated Herb Shop for purchasing recommended herbal formulas.
- **Digital Twin**: Live visualization of organ health status.
- **Multi-Language Support**: Full support for English, Chinese (Simplified), and Bahasa Malaysia.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: Radix UI, Framer Motion
- **AI**: Gemini 2.0 Flash / Pro (via Vercel AI SDK)
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth

## Getting Started

First, install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
