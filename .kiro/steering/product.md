# Product Overview

Sihat TCM is an AI-powered Traditional Chinese Medicine (TCM) diagnosis system that combines ancient TCM wisdom with modern AI technology. The platform provides accessible, personalized, and multi-lingual health consultations through both web and mobile applications.

## Core Product Features

- **Four Pillars Diagnosis**: AI-powered analysis using Observation (Tongue/Face), Listening (Voice), Inquiry (Chat), and Palpation (Pulse)
- **Personalized TCM Reports**: Detailed pathomechanism analysis and holistic recommendations
- **AI Meal Planner**: Constitution-based 7-day meal plans with shopping lists
- **Health Tracking**: Integration with IoT devices and health apps (Apple Health, Google Fit)
- **Gamification**: Qi Dose exercises, Qi Garden virtual herb growing, achievement system
- **Family Care**: Multi-profile management for entire families
- **Multi-Language Support**: English, Chinese (Simplified), and Bahasa Malaysia

## Target Users

- **Primary**: Individuals seeking TCM health guidance and wellness advice
- **Secondary**: Healthcare practitioners using TCM diagnostic tools
- **Tertiary**: Families managing health profiles for multiple members

## Platform Architecture

- **Web Application**: Next.js-based responsive web app
- **Mobile Application**: Expo/React Native cross-platform mobile app
- **Backend**: Supabase (PostgreSQL) with real-time synchronization
- **AI Integration**: Google Gemini models with intelligent routing and fallback

## Key Differentiators

- Advanced AI model routing based on request complexity
- Medical safety validation for all recommendations
- Personalization engine that learns from user preferences
- Cross-platform synchronization between web and mobile
- Comprehensive TCM knowledge base with modern UX/UI
